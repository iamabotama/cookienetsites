## Usage Notes

- **"Backed at over 100%"** is intentionally unspecific so the message stays true as the ratio drifts. The exact figure with methodology lives in whitepaper Section 3.4; link there if anyone asks.
- **Do not add claims like "no team allocation" or "no insider wallets."** The project has a disclosed 10% team operations allocation (whitepaper Section 3.2); the marketing angle is full disclosure, not absence of allocations.
- **Dexscreener link:** add the sCOOK/SOL pair URL under the CA block (label "Dex") once confirmed.
- Reusable taglines: "We don't ask for trust. We publish addresses." / "Every cookie accounted for. Every exit backed."
